dvn-vagrant setup notes

install virtualbox
install vagrant
clone install demo @ https://github.com/dvn/dvn-install-demo
follow readme

Add to Vagrantfile:
	config.vm.forward_port 443, <something>
(may need to modify port 80 as well)

Add to /etc/sysconfig/iptables:
	-A INPUT -m state --state NEW -m tcp -p tcp --dport 443 -j ACCEPT
